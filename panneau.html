VIN EXPERT ATTT
Application web progressive (PWA) pour l'identification, la vérification et la traçabilité des numéros de châssis VIN — Agence Technique de Traitement des Titres de Transport
🔍

Identification VIN

  • Saisie manuelle — entrez le VIN (17 caractères) directement
  • Scan photo — photographiez la plaque VIN, l'OCR extrait le code automatiquement (Tesseract.js)
  • Décodage automatique — via l'API NHTSA (base de données officielle US/EU)
  • OCR local — lecture et correction du VIN entièrement sur l'appareil avec Tesseract.js
📄

Rapports PDF

  • Génération instantanée — rapport PDF officiel ATTT après chaque identification
  • Contenu — marque, modèle, année, moteur, pays de fabrication, statut validité
  • Signature agent — nom de l'agent, date, heure et numéro de rapport
  • Sauvegarde Firestore — chaque rapport est archivé dans la base de données
👥

Gestion agents

  • Inscription sur invitation — code unique ATT-XXXXXXXX requis pour s'inscrire
  • Validation admin — tout nouveau compte doit être approuvé manuellement
  • Rôles — Admin, Agent actif, En attente, Désactivé
  • Email invitation Gmail — l'agent envoie l'invitation depuis son propre compte Gmail
🚀

Déploiement

  • GitHub Pages — hébergement gratuit, accessible sur tous les appareils
  • Premier déploiement — dans l'onglet Publication, section déploiement initial
  • Mises à jour — publiez une nouvelle version avec notes de mise à jour
  • Service Worker PWA — mise à jour automatique notifiée à tous les appareils

📋 Procédure d'utilisation — étape par étape
1️⃣

Premier déploiement

  • Lancez ATTT.bat → option 2 pour ouvrir le panneau admin
  • Connectez-vous avec le compte dta.attt.2026@gmail.com
  • Allez dans ⚙️ Configuration → saisissez votre token GitHub → Sauvegarder
  • Allez dans 🚀 Publication → section "Premier déploiement" → sélectionnez les fichiers du dossier
  • Cliquez DÉPLOYER et attendez la confirmation
2️⃣

Ajouter un agent

  • Dans l'app (index.html), allez dans Mon compte → Invitations
  • Cliquez Envoyer par email, saisissez l'email du futur agent
  • Gmail s'ouvre avec tout pré-rempli — cliquez ENVOYER
  • Le destinataire clique le lien dans l'email et s'inscrit avec le code
  • Dans ce panneau → 👥 Agents → approuvez le compte
3️⃣

Identifier un VIN

  • Ouvrez l'application sur mobile ou PC (lien ci-dessous)
  • Connectez-vous avec votre compte agent
  • Tapez le VIN (17 caractères) ou cliquez 📷 Scanner
  • Photographiez la plaque — l'OCR extrait automatiquement le VIN
  • Cliquez IDENTIFIER → consultez les résultats
  • Téléchargez le rapport PDF officiel
4️⃣

Publier une mise à jour

  • Modifiez les fichiers sources localement
  • Ouvrez le panneau → 🚀 Publication
  • Incrémentez la version (ex: 1.0.0 → 1.1.0)
  • Rédigez les notes de mise à jour
  • Cliquez PUBLIER → les agents reçoivent une notification de mise à jour

🔧 Informations techniques
☁️

Infrastructure

  • Backend — Firebase (Google) : Auth + Firestore
  • Hébergement — GitHub Pages (gratuit, HTTPS)
  • OCR — Tesseract.js local (gratuit, sans service cloud)
  • Données VIN — API NHTSA (base officielle)
  • PDF — jsPDF 2.5.1 (génération côté client)
🔒

Sécurité

  • Authentification — Firebase Auth (email + mot de passe)
  • Contrôle d'accès — rôles vérifiés côté Firestore
  • Invitations — codes à usage unique, expiration automatique
  • Token GitHub — stocké dans Firestore (chiffré), jamais en localStorage
  • Connexion secours — accès admin local en cas d'urgence
📡

Navigateurs compatibles

  • Chrome 90+ — recommandé (PWA, caméra, PDF)
  • Edge 90+ — compatible
  • Firefox 88+ — compatible (sans installation PWA)
  • Safari 14+ — iOS iPhone/iPad (PWA via Safari)
  • Samsung Internet — Android
